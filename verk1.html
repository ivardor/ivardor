Hýsing vefsíðu á GitHub

"You need a website"

Nú veit ég ekki hvar þú hefur verið á internetinu undanfarna mánuði en svo virðist sem VPN-auglýsinga aldan sé nú að líða undir lok og við tekur YOU NEED A WEBSITE stormurinn. Það er jú ekki seinna vænna að undirbúa sig undir fall facebook samsteypunar og tryggja tilvist sína á netinu og sjálfsagt að hefja þá vegferð með uppsetningu á eigin vefsíðu.

Undirbúningur

Í þessari uppskrift er ekki gert ráð fyrir neinni þekkingu á virkni hvorki git umhverfinu né GitHub. Ef þér tókst að komast á internetið getur þú afrekað það að hýsa vefsíðu á GitHub. Til hamingju, þú ert tilbúin/nn!

Stofnun GitHub aðgangs

Ef þú átt hann ekki nú þegar þarft þú að stofna GitHub aðgang. Leggðu leið þína beinustu leið inn á GitHub og smelltu á "Sign up" hnappinn. Eftir að hafa fyllt út allar helstu upplýsingar
